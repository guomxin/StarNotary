## StarNotary
一个声明星球占有权的基于以太坊的DApp
### 运行环境
Truffle v5.1.33 (core: 5.1.33)  
Solidity v0.5.16 (solc-js)  
Node v10.16.1  
Web3.js v1.2.1  
### 预先安装
- node (v10)
- Metamask
### 运行方式
1. 安装truffle  
`npm install truffle -g`
2. 当前目录运行  
`truffle develop`
3. 在truffle console里面运行：
   - compile
   - migrate --reset
4. 进入app目录运行：
   - npm install
   - npm run dev
5. 浏览器访问http://localhost:8080
